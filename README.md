# QR Code Component

This is a solution to the **QR Code Component challenge** from Frontend Mentor.

## Overview

The project is a simple, responsive QR code card built with **HTML and CSS**, following the provided design specifications.

### Features
- Mobile-first responsive layout
- Clean, centered card design
- Accessible markup
- Uses Google Fonts (Outfit)

## Built With
- HTML5
- CSS3 (Flexbox)
- Mobile-first workflow

## What I Learned
- Structuring components using container-based layouts
- Applying responsive design with media queries
- Translating design specs into CSS
- Using Git and GitHub for version control
